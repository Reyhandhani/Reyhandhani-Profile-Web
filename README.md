## 🚀 Getting Started

### Prerequisites

Make sure you have the following installed:
- [Git](https://git-scm.com/)
- A modern web browser or your preferred runtime environment

### Installation

```bash
# Clone the repository
git clone https://github.com/Reyhandhani/Reyhandhani-Profile-Web.git

# Navigate into the project directory
cd Reyhandhani-Profile-Web
```

### Running the Project

Open `index.html` in your browser, or if using a local dev server:

```bash
# Example using VS Code Live Server or any static file server
npx serve .
```
